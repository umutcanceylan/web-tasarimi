CSS ekleme yöntemleri

  1. Yerel CSS

    Yerel CSS, stil etiketinin parametre olarak yazıldığı css ekleme yöntemidir.

  2. Dahili CSS

    Dahili CSS etiketleri head bloğuna style etiketleri arasına yazılır.

  3. Harici CSS

    Harici CSS eklemek için css dosyası oluşturulur.

Seçiciler

ID Seçici

#idadi verilen css özelliği, ilgili HTML etiketininde ID ile çağırılarak kullanılır.

Class Seçici

.classadi ile verilen CSS özelliği, ilgili HTML etiketininde class ile çağırılarak kullanılır.
